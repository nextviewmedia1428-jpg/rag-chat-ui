## Customer Support Workflow

### Standard Flow

1. Customer contacts support via app, phone, chat, or email
2. Support agent verifies product registration and warranty status
3. If under warranty → book in-home technician visit at no charge (Year 1) or quote applicable charges (Year 2+)
4. If out of warranty → quote service charge before dispatch
5. Technician visits within SLA, diagnoses issue, and repairs or orders parts
6. Job card raised and customer signs off after repair
7. Customer receives feedback call within 48 hours

### Out-of-Warranty Repair

- Minimum diagnostic charge: ₹350 (applicable even if repair not taken forward)
- Diagnostic charge adjusted against repair bill if repair is authorized
- Part and labour quote shared before work begins
- Customer confirmation required before any chargeable repair

## Error Code Reference

### Front-Load Washing Machine Error Codes

| Error Code | Meaning | Customer Action |
|---|---|---|
| E1 | Water inlet timeout — not filling within set time | Check water supply tap is open; check inlet hose is not kinked |
| E2 | Drainage timeout — not draining within set time | Check drain hose is not blocked or kinked; check filter |
| E3 | Motor overload or drive fault | Stop machine, wait 15 minutes, restart. If recurring, call service |
| E4 | Door not locked or door switch fault | Ensure door is properly closed; call service if persists |
| E5 | Over-suds / excess detergent detected | Run rinse cycle without detergent; reduce detergent quantity next wash |
| E6 | Water temperature sensor fault | Do not use machine; call service |
| E7 | Unbalanced load | Open door, redistribute laundry, restart |
| E9 | Water level sensor fault | Do not use machine; call service |
| F1 | Communication fault between control board and motor | Restart machine; if recurring, call service |

## Common Troubleshooting Guides

### Washing Machine — Not Starting
1. Confirm power supply is connected and socket is live
2. Confirm door is fully closed (machine will not start if door is ajar)
3. Check if child lock is activated — hold Child Lock button for 3 seconds to deactivate
4. Check if water supply tap is open
5. If display shows an error code, refer to error code table above
6. If no display at all, check fuse/circuit breaker

### Washing Machine — Excessive Vibration or Noise
1. Ensure machine is level — adjust feet until machine does not rock
2. Remove transit bolts if newly installed (4 bolts at back of machine)
3. Check load is evenly distributed in drum
4. Ensure load weight is within rated capacity
5. Check for foreign objects (coins, bra wires) in drum or pump filter

### Air Conditioner — Not Cooling
1. Check remote temperature setting — ensure it is set lower than room temperature
2. Clean air filter — dirty filter reduces cooling by up to 30%
3. Check that outdoor unit is not blocked by debris or shrubs
4. Check if cooling mode is selected (not fan-only or auto)
5. Check if room doors/windows are closed
6. If cooling is significantly reduced and filter is clean, refrigerant may need topping up — call service

### Refrigerator — Not Cooling
1. Check temperature setting — fridge section should be set to 3–4, freezer to maximum
2. Check door gaskets are sealing properly (close door on a piece of paper — if paper slips out easily, gasket needs replacement)
3. Ensure vents inside are not blocked by overfilled shelves
4. Check condenser coils at back are dust-free
5. If compressor does not start (no humming sound), call service

### Dishwasher — Dishes Not Clean
1. Check rinse aid level and top up if empty
2. Ensure spray arms are not blocked — remove and rinse under running water
3. Check water temperature — dishwasher requires minimum 55°C supply for best results
4. Do not pre-rinse plates excessively — sensors need soil to activate correct wash level
5. Use correct quantity of detergent for your water hardness level
